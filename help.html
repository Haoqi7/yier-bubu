💖 关于一二与布布

一二与布布是一款可爱的桌面宠物软件，内置物理引擎和行为模拟系统，布布和一二会在你的桌面上自由活动、互相追逐、陪伴你工作和学习。

一二：活泼好动、爱撒娇、喜欢捉迷藏和爬墙，偶尔会自己偷吃零食。

布布：安静沉稳、爱编码、喜欢看书和喝咖啡，经常在屏幕角落安静工作。

🎮 交互方式
  • 右键点击一二布布或空白区域：呼出操作菜单
  • 左键拖拽：抓住一二布布移动（有惯性释放，松手会掉落）
  • 喂食：提供多种食物，提升饱腹度和心情
  • 玩耍：使用玩具互动，提升心情和亲密度
  • 爱抚：摸摸增加亲密度
  • 一二布布之间会自动互相追逐、打招呼、早安问候等
📊 查看状态

通过 托盘菜单 → 查看状态 或右键菜单打开，实时显示一二布布四项核心属性：

  • 🍖 饱腹度：随时间下降，为零时一二布布会饥饿讨食，需要及时喂食
  • 💕 心情值：互动（喂食/玩耍/爱抚）可提升，心情低落时一二布布会难过
  • ⚡ 精力值：活动时消耗，低精力时一二布布会打哈欠、想睡觉
  • 💗 亲密度：通过爱抚、玩耍等互动逐步累积
  • 同时显示 天气信息（模拟天气系统，影响一二布布行为）
  • 显示一二布布的当前状态（发呆/走路/睡觉等）
📖 一二布布日记

通过 托盘菜单 → 一二布布日记 或右键菜单打开，自动记录你与一二布布的互动点滴：

  • 每条记录包含 图标 + 事件描述 + 时间戳
  • 自动记录的事件类型：起床问候、吃饭、玩耍、爱抚、情侣互动、拖拽、掉落、睡眠等
  • 最近 50 条记录按时间倒序显示
  • 数据以 JSON 文件本地持久化存储，不会丢失
  • 日记会根据一二布布的个性生成不同的文字描述（一二和布布的风格不同）
🎞️ GIF 状态管理面板

通过 托盘菜单 → GIF 状态 打开，用于管理所有 GIF 素材库：

📋 功能说明
  • 面板包含三个标签页：一二、布布、待分配
  • 上传 GIF：点击按钮选择本地 .gif 文件上传到当前标签页
  • 上传时自动根据当前模式判断归属：仅一二模式上传给一二，仅布布上传给布布，双人模式上传到待分配
  • 触发概率：每个 GIF 可设置 1%~20% 的触发概率
  • 拖拽上传：直接将 GIF 文件拖拽到一二布布身上可快速分配
  • 清空：支持一键清空当前标签页所有 GIF（需三次确认）
  • 每个 GIF 卡片显示预览、归属、上传时间和触发概率
  • 删除 GIF 时会自动清除所有相关绑定
🎭 GIF 绑定管理

通过 托盘菜单 → GIF 绑定管理 打开，将 GIF 素材绑定到具体行为状态：

📋 功能说明
  • 面板分两个标签：🤍 一二 和 🤎 布布，各自独立管理
  • GIF 库：显示当前一二布布的所有可用 GIF，可在此上传新 GIF 或删除
  • 状态绑定区：列出该一二布布的所有状态（已绑定在上、未绑定在下）
  • 点击状态卡片展开详情，可添加/删除绑定、调整概率
📋 操作流程
  • ① 上传 GIF → ② 展开目标状态卡片 → ③ 点击「添加 GIF」选择素材 → ④ 点击「下一步」调整概率 → ⑤ 点击「确认绑定」
  • 支持为同一状态绑定 多个 GIF，触发时按概率随机选择
  • 多个 GIF 的概率自动归一化，可手动调节每个 GIF 的权重
  • 解绑：点击绑定项的删除按钮，二次确认后解绑，该状态自动恢复默认动画
  • 立即生效：绑定/解绑后，如果一二布布正在该状态，动画立即替换
⚙️ 其他功能
  • 设置：切换窗口模式（全屏/手动划分）、开机自启动、鼠标穿透模式
  • 专注模式：一二布布暂时安静不干扰，专注工作/学习
  • 切换模式：可选择「一二与布布」「仅一二」「仅布布」三种模式
  • 所有功能入口在系统托盘（任务栏小图标）右键菜单中
右键 呼出操作菜单
💡 小贴士
  • 一二布布会按真实作息自动吃饭、睡觉、起床问候（早 6~9 点）
  • 早餐/午餐/晚餐有固定时间段，错过会饿肚子
  • 长时间不喂食，一二布布会变得饥饿和难过，头顶出现 🍖 图标
  • 拖拽到高处一二布布会害怕，头顶出现恐惧表情
  • 一二布布靠近时会触发情侣互动（追逐、夸夸、早安问候等）
  • 一二布布心情好时会自己跳舞、看电视、打滚
