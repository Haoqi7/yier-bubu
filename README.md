#  一二与布布 - 桌面宠物

一款可爱的桌面宠物软件，内置物理引擎和行为模拟系统，「一二」和「布布」会在你的桌面上自由活动、互相追逐、陪伴你工作和学习。

## 🌟 特色功能

### 🐾 一二布布
- **一二**：活泼好动、爱撒娇、喜欢捉迷藏和爬墙，偶尔会自己偷吃零食
- **布布**：安静沉稳、爱编码、喜欢看书和喝咖啡，经常在屏幕角落安静工作

### 🎮 丰富的交互
- **右键菜单**：喂食、玩耍、爱抚、查看状态、打开日记
- **拖拽**：抓住移动，有物理惯性，松手后会掉落
- **自动互动**：一二布布之间会互相追逐、打招呼、早安问候
- **情侣互动**：一二布布靠近时触发特殊互动（追逐、夸夸、早安问候等）

### 📊 查看状态
通过托盘菜单或右键菜单打开，实时显示每只宠物的核心属性：
- **🍖 饱腹度**：随时间下降，为零时宠物会饥饿讨食
- **💕 心情值**：互动可提升，心情低落时宠物会难过
- **⚡ 精力值**：活动时消耗，低精力时会打哈欠、想睡觉
- **💗 亲密度**：通过爱抚、玩耍等互动逐步累积
- 同时显示天气信息和宠物当前状态

### 📖 宠物日记
自动记录你与宠物的互动点滴：
- 每条记录包含图标、事件描述和时间戳
- 自动记录事件：起床问候、吃饭、玩耍、爱抚、情侣互动、拖拽、掉落、睡眠等
- 最近 50 条记录按时间倒序显示
- 数据以 JSON 文件本地持久化存储
- 一二和布布会生成不同风格的日记文字

### 🎞️ GIF 状态管理
管理所有 GIF 素材库：
- 三个标签页：**一二**、**布布**、**待分配**
- 上传本地 .gif 文件，自动根据当前模式判断归属
- 待分配 Tab 存放无法自动判断归属的 GIF，可手动分配
- 每个 GIF 可设置 1%~20% 的触发概率
- 支持拖拽 GIF 到宠物身上快速分配
- 清空当前标签页（需三次确认）
- 删除 GIF 时自动清除所有相关绑定

### 🎭 GIF 绑定管理
将 GIF 素材绑定到具体行为状态：
- 两个标签独立管理：**一二** 和 **布布**
- GIF 库显示当前宠物的所有可用 GIF
- 状态绑定区列出该宠物的所有状态（已绑定在上、未绑定在下）
- **操作流程**：上传 GIF → 展开目标状态 → 添加 GIF → 下一步调整概率 → 确认绑定
- 支持为同一状态绑定多个 GIF，触发时按概率随机选择
- 多个 GIF 的概率自动归一化，可手动调节权重
- 解绑需二次确认，解绑后自动恢复默认动画
- 绑定/解绑后立即生效

### 🧠 真实行为模拟
- **作息系统**：自动吃饭、睡觉、打哈欠、洗澡（早 6~9 点起床问候）
- **情绪系统**：心情、饱腹度、精力、亲密度随时间变化
- **物理引擎**：重力、碰撞、行走、跳跃、掉落
- **天气系统**：模拟天气影响宠物行为
- **个性行为**：一二偷吃零食、布布安静编码

### ⚙️ 灵活设置
- 窗口模式：全屏 / 自定义区域
- 开机自启动
- 鼠标穿透模式
- 专注模式
- 切换模式：一二与布布 / 仅一二 / 仅布布

## 📦 安装

### 环境要求
- Node.js >= 16
- npm >= 8

### 开发模式
```bash
# 克隆项目
git clone https://github.com/Haoqi7/yier-bubu.git
cd yier-bubu

# 安装依赖
npm install

# 启动开发模式
npm start
```

### 打包发布
```bash
# Windows
npm run build:win

# macOS
npm run build:mac
```

## 📁 项目结构

```
yier-bubu/
├── assets/                # 资源文件（帧动画、GIF 等）
│   ├── yier/              # 一二的帧动画
│   └── bubu/              # 布布的帧动画
├── renderer/              # 前端代码
│   ├── app.js             # 主应用逻辑（物理引擎、行为模拟、交互系统）
│   ├── gif-manager.js     # GIF 素材库管理
│   ├── gif-state-panel.js # GIF 状态绑定面板
│   └── styles.css         # 全局样式
├── config.js              # 配置文件（菜单、状态、作息、个性等）
├── main.js                # Electron 主进程（窗口管理、托盘菜单）
├── preload.js             # 预加载脚本（IPC 通信桥接）
├── index.html             # 主界面
├── settings.html          # 设置界面
├── help.html              # 帮助界面
└── package.json           # 项目配置
```
